About Notely

Notely saves notes locally in your browser using IndexedDB (no account required). Notes auto-save as you type and persist on the same device & browser. Clearing browser data will remove your notes.

Use the toolbar or type markdown-like shortcuts directly in the editor. The exercises below help you practice those shortcuts.

Headings

Type # at the start of a line, then press space. The # will be removed and the line becomes a level-1 heading.

Heading (H1)
Bold

Bold

Type **your text** and press space. The stars will be hidden and the text will become bold.

Italic

Wrap text with single asterisks: *italic*. After pressing space, the asterisks are removed and text becomes italic.

Italic
Dash list

Dash list

Start a line with - then press space. That line will become a bullet list item (dash list).

Dot list (asterisk)

Start a line with * then press space. This also creates an unordered list (dot-style).

Dot list (*)
Numbered list

Numbered list

Start a line with 1. (or any number + dot) then press space. That line becomes a numbered list item.
